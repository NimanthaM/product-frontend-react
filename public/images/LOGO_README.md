Place your Aahaas logo in this folder with the filename: **`aahaas-logo.png`**

### Logo Requirements:
- **Filename**: `aahaas-logo.png` (or `.jpg`, `.svg`)
- **Recommended Size**: Height of 40-80 pixels (width will auto-adjust)
- **Format**: PNG (with transparent background recommended), JPG, or SVG
- **Background**: Transparent PNG works best for professional look
- **File Size**: Keep under 200KB

### Tips:
- Use PNG format with transparent background for best results
- Logo should be clear and readable at small sizes
- Square or horizontal logos work best
- The logo will be displayed at 40px height by default

### Current Setup:
The Header component expects the logo at:
```
/images/aahaas-logo.png
```

### Alternative Formats:
If you use a different format, update the file extension in `src/components/Header.jsx`:
```javascript
src="/images/aahaas-logo.svg"  // For SVG
src="/images/aahaas-logo.jpg"  // For JPG
```

---

**Note**: The logo will appear in the navbar before the word "Ecommerce".
